# Warehouse Management System (RGR Project)

Веб-застосунок "Система управління складом", розроблений в рамках розрахунково-графічної роботи. Проєкт реалізує облік товарів з розмежуванням прав доступу.

## 📋 Функціонал

### Ролі користувачів:
* **Гість:** Може переглядати головну сторінку, входити в систему або реєструватися.
* **Користувач (USER):** Може переглядати каталог товарів (Grid layout) з фото та цінами.
* **Адміністратор (ADMIN):** Має повний доступ до управління товарами (CRUD): додавання, редагування, видалення.

### Особливості:
* **Безпека:**
    * Паролі користувачів шифруються алгоритмом **Base64** (вимога ТЗ).
    * Захист від **SQL Injection** завдяки використанню Spring Data JPA.
    * Валідація форм реєстрації (унікальність логіну, підтвердження пароля).
* **Інтерфейс:**
    * Адаптивний дизайн на базі **Bootstrap 5**.
    * Відображення товарів у вигляді карток.
    * Підтримка посилань на зображення.

## 🛠 Технологічний стек

* **Backend:** Java 21, Spring Boot 3.4.1 (Web, Security, Data JPA).
* **Database:** PostgreSQL 15 (Docker).
* **Frontend:** Thymeleaf, Bootstrap 5.
* **Build Tool:** Maven.
